# Clawb AI Agency Landing Page

Professional agency website for **Clawb AI Agency** built with **React + Vite + Tailwind CSS**.

## Features
- Dark, glass-morphism UI with violet/indigo accents
- Smooth-scroll navigation and responsive layout
- Hero, Services, Portfolio, Pricing, Process, About, Footer sections
- Scroll-triggered fade-in animations

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
